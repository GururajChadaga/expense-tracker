# Expense Tracker

Expense tracker is a react app that lets the user track their expenses by allowing them to add an expense with the fields of Title, amount and date of expinditure.

# PreRequisites
[Node.js](https://www.google.com "https://nodejs.org/en/") must be installed.


# How to run on development server

1. Clone the repo and get into the directory
```
git clone git@github.com:GururajChadaga/expense-tracker.git
cd expense-tracker
```
2. Install dependencies(has to be done only once after cloning the repo)
```
npm install
``` 
3. Start the app  
```
npm start
```
The app now runs on localhost:3000 by default.
